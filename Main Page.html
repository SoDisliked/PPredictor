PPredictor

Stock prices using API and TensorFlow package with MACD function

The Machine Learning, MACD and Volatility methods help to see the prediciton of prices into different markets, especially related to stocks.

Disclaimer: Stock prices can change and can be hard to estimate as the methods may eronate and a sudden sell or buy may change the volume of an index. The results can not be therefore 100% accurate.

4 parts to explain the usage of the website/app:

  • gather the historical volume data of stocks
  • Usage of Neural%20Network system built on a Machine Learning system.
  • training model that uses the configs and the variables inputs thanks to TensorFlow, PyTorch, Pandas, NumPy
  • predictions can be made
